# :black_nib: 简单找房

> 实时爬取豆瓣租房信息，爬取到对应关键字的帖子主动推送用户，节省你宝贵的时间。

## 模块
- 前端：HTML + JQuery +Ajax
- 后端：SpringBoot + SpringData
- 爬虫：Python3.6
